code . --> permet d'ouvrir VSCode depuis le terminal, dans le répertoire où on se trouve.

Si on commence par le dépot local

  • Se placer sur le dossier
    cd Documents/WCSquests/GitGithub2-GitEnLocal/hello-world
  • configurer Git pour qu'il génère une branche main par défaut
    git config --global init.defaultBranch main
  • Initialiser le dossier en local
    git init
  • Lier le dépôt local avec le dépôt distant
    git remote add origin git@github.com:Tadkozh/my-super-website.git
  • git add qqchose : placer la modif en "stage"
  • git -m "truc" : faire un commit avec son message
  • Savoir sur quelle branche on est
    git branch --show-current
  • Changer le nom de la branche master ou on se trouve en main
    git branch -m main
  • Pour le premmier push vers le repo distant, il faut accorder les deux branches main, la locale et la distante
    git push --set-upstream origin main

Si on commence par le dépot distant

  • Importer un projet à partir d'un dépôt distant (si on prend un projet en cours de route)
    git clone git@github.com:Tadkozh/my-super-website.git

Echanges entre les deux dépôts

  • "pousser" le code local vers le repo distant
    git push origin main
  • "récupérer" en local la dernière version du code distant
    git pull origin main

Faire un commit

  • git branch --show-current : savoir sur quelle branche on est
  • git status : savoir où on en est dans nos modifs
  • git add qqchose : placer la modif en "stage"
  • git -m "truc" : faire un commit avec son message
  • git log : obtenir un historique du projet
