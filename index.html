Si on commence par le dépot local

  • se placer sur le dossier
    cd Documents/WCSquests/GitGithub2-GitEnLocal/hello-world
  • Initialiser le dossier en local
    git init
  • Lier le dépôt local avec le dépôt distant
    git remote add origin git@github.com:Tadkozh/my-super-website.git

Si on commence par le dépot distant

  • Importer un projet à partir d'un dépôt distant (si on prend un projet en cours de route)
    git clone git@github.com:Tadkozh/my-super-website.git

Echanges entre les deux dépôts

  • "pousser" le code local vers le repo distant
    git push origin main
  • "récupérer" en local la dernière version du code distant
    git pull origin main

Faire un commit

  • "git status : savoir où on en est dans nos modifs
  • "git add qqchose : placer la modif en "stage"
  • "git -m : faire un commit avec son message
